Workbench: CFD Automation

An Automated CFD Protocol Using .jou Script

CFD Automation: This is an automated protocol for simulation and post-processing of a single run (specific AoA) CFD study for a 2D airfoil (or any other 2D object). These scripts are designed for the Spalart Allmaras governing equation, but future updates will add other governing equations and associated settings for turbulence modeling and flow seperation. Python and the built in 'Journal' feature (.jou) of ANSYS were used to automate the workflow.

The CFD simulation was done for a NACA0012 (Chord Length = 1m) airfoil. Please consult the documentation (instruction manual) and the Git repository for the journal and Python scripts.

Additional scripts for turbulence modeling for post-stall conditions will be shared here soon.
